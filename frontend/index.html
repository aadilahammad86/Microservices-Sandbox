The Blueprint

Under the hood, this project runs on three distinct containers interacting via a custom Docker bridged network.

Frontend (Nginx) Serves the UI & Proxies Traffic
Backend (Node.js) Express REST API
Database (PostgreSQL) Persistent Data Volume

Automated Delivery

Code pushed to GitHub triggers a GitHub Actions workflow that runs tests, builds the Docker images, and pushes them to the GitHub Container Registry (GHCR).

1 Git Push
2 Run Tests
3 Build Images
4 Push to Registry
